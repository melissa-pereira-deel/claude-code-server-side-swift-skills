# Scripts Reference

Complete ci_scripts/ reference including all lifecycle hooks, environment variables, and common patterns.

## Script Lifecycle

```
1. Repository cloned
2. ci_post_clone.sh        ← Install tools, dependencies
3. xcodebuild resolves SPM packages
4. ci_pre_xcodebuild.sh   ← Pre-build setup, code generation
5. xcodebuild runs (build/test/archive)
6. ci_post_xcodebuild.sh  ← Post-build notifications, uploads
```

## Script Setup

### Directory Structure

```
project-root/
├── ci_scripts/
│   ├── ci_post_clone.sh
│   ├── ci_pre_xcodebuild.sh
│   └── ci_post_xcodebuild.sh
├── MyApp.xcodeproj
└── ...
```

### Script Permissions

```bash
# Scripts MUST be executable
chmod +x ci_scripts/ci_post_clone.sh
chmod +x ci_scripts/ci_pre_xcodebuild.sh
chmod +x ci_scripts/ci_post_xcodebuild.sh

# Verify in git
git ls-files -s ci_scripts/
# Should show 100755 (executable)

# If not executable, fix:
git update-index --chmod=+x ci_scripts/ci_post_clone.sh
```

### Script Header

```bash
#!/bin/bash
# Always use bash, always fail on errors
set -e

echo "=== Running $(basename "$0") ==="
echo "Action: ${CI_XCODEBUILD_ACTION:-none}"
echo "Branch: ${CI_BRANCH:-none}"
echo "Commit: ${CI_COMMIT:-none}"
```

## ci_post_clone.sh

### Purpose
Runs immediately after the repository is cloned. Use for installing tools and dependencies.

### Template

```bash
#!/bin/bash
set -e

echo "=== Post Clone Script ==="

# ──────────────────────────────────────────────
# 1. Install Homebrew dependencies
# ──────────────────────────────────────────────
if command -v brew &> /dev/null; then
    echo "Installing Homebrew dependencies..."
    brew install swiftlint sourcery swiftgen
fi

# ──────────────────────────────────────────────
# 2. Install Ruby dependencies (for Fastlane, CocoaPods)
# ──────────────────────────────────────────────
if [ -f "Gemfile" ]; then
    echo "Installing Ruby gems..."
    gem install bundler
    bundle install
fi

# ──────────────────────────────────────────────
# 3. Install Node.js dependencies (for tools)
# ──────────────────────────────────────────────
if [ -f "package.json" ]; then
    echo "Installing Node.js dependencies..."
    npm install
fi

# ──────────────────────────────────────────────
# 4. CocoaPods (if used)
# ──────────────────────────────────────────────
if [ -f "Podfile" ]; then
    echo "Installing CocoaPods..."
    pod install
fi

# ──────────────────────────────────────────────
# 5. Write build configuration
# ──────────────────────────────────────────────
echo "Build Number: $CI_BUILD_NUMBER"
echo "Commit: $CI_COMMIT"

echo "=== Post Clone Complete ==="
```

### SPM Dependencies with Private Repos

```bash
#!/bin/bash
set -e

# For private SPM dependencies, configure git credentials
# Use Xcode Cloud's built-in source control authentication
# Or use a personal access token from environment secrets

if [ -n "$GITHUB_TOKEN" ]; then
    echo "Configuring git authentication for private packages..."
    git config --global url."https://x-access-token:${GITHUB_TOKEN}@github.com/".insteadOf "https://github.com/"
fi
```

## ci_pre_xcodebuild.sh

### Purpose
Runs before each xcodebuild action (build, test, archive). Use for code generation and environment-specific setup.

### Template

```bash
#!/bin/bash
set -e

echo "=== Pre Xcodebuild Script ==="
echo "Action: $CI_XCODEBUILD_ACTION"

# ──────────────────────────────────────────────
# 1. Code generation (SwiftGen, Sourcery, etc.)
# ──────────────────────────────────────────────
if command -v swiftgen &> /dev/null; then
    echo "Running SwiftGen..."
    swiftgen
fi

if command -v sourcery &> /dev/null; then
    echo "Running Sourcery..."
    sourcery
fi

# ──────────────────────────────────────────────
# 2. Write build configuration file
# ──────────────────────────────────────────────
CONFIG_FILE="Sources/App/BuildConfig.swift"
cat > "$CONFIG_FILE" <<EOF
// Auto-generated by ci_pre_xcodebuild.sh
// DO NOT EDIT
enum BuildConfig {
    static let buildNumber = "$CI_BUILD_NUMBER"
    static let commitHash = "${CI_COMMIT:0:7}"
    static let branch = "$CI_BRANCH"
    static let isCI = true
}
EOF
echo "Generated $CONFIG_FILE"

# ──────────────────────────────────────────────
# 3. Environment-specific configuration
# ──────────────────────────────────────────────
case "$CI_XCODEBUILD_ACTION" in
    "archive")
        echo "Archive build — production configuration"
        # Write production API endpoints
        ;;
    "test"|"build-for-testing")
        echo "Test build — test configuration"
        # Write staging API endpoints
        ;;
esac

# ──────────────────────────────────────────────
# 4. SwiftLint (on test/build actions only)
# ──────────────────────────────────────────────
if [[ "$CI_XCODEBUILD_ACTION" != "archive" ]]; then
    if command -v swiftlint &> /dev/null; then
        echo "Running SwiftLint..."
        swiftlint lint --reporter github-actions-logging || true
    fi
fi

echo "=== Pre Xcodebuild Complete ==="
```

## ci_post_xcodebuild.sh

### Purpose
Runs after xcodebuild completes. Use for notifications, metrics, and artifact processing.

### Template

```bash
#!/bin/bash
set -e

echo "=== Post Xcodebuild Script ==="
echo "Action: $CI_XCODEBUILD_ACTION"
echo "Exit Status: ${CI_XCODEBUILD_EXIT_CODE:-unknown}"

# ──────────────────────────────────────────────
# 1. Slack notification
# ──────────────────────────────────────────────
send_slack_notification() {
    local status=$1
    local color=$2
    local message=$3

    if [ -z "$SLACK_WEBHOOK_URL" ]; then
        echo "No Slack webhook configured, skipping notification"
        return
    fi

    curl -s -X POST "$SLACK_WEBHOOK_URL" \
        -H "Content-Type: application/json" \
        -d "{
            \"attachments\": [{
                \"color\": \"$color\",
                \"title\": \"$CI_PRODUCT — $CI_WORKFLOW\",
                \"text\": \"$message\",
                \"fields\": [
                    {\"title\": \"Branch\", \"value\": \"$CI_BRANCH\", \"short\": true},
                    {\"title\": \"Build\", \"value\": \"#$CI_BUILD_NUMBER\", \"short\": true},
                    {\"title\": \"Action\", \"value\": \"$CI_XCODEBUILD_ACTION\", \"short\": true},
                    {\"title\": \"Status\", \"value\": \"$status\", \"short\": true}
                ]
            }]
        }"
}

if [ "${CI_XCODEBUILD_EXIT_CODE:-1}" -eq 0 ]; then
    send_slack_notification "✅ Success" "#36a64f" "Build succeeded"
else
    send_slack_notification "❌ Failed" "#ff0000" "Build failed"
fi

# ──────────────────────────────────────────────
# 2. Process test results
# ──────────────────────────────────────────────
if [[ "$CI_XCODEBUILD_ACTION" == "test"* ]] && [ -d "$CI_RESULT_BUNDLE_PATH" ]; then
    echo "Processing test results..."
    xcrun xcresulttool get --format json \
        --path "$CI_RESULT_BUNDLE_PATH" > test_results.json 2>/dev/null || true
fi

# ──────────────────────────────────────────────
# 3. Archive info
# ──────────────────────────────────────────────
if [[ "$CI_XCODEBUILD_ACTION" == "archive" ]] && [ -d "$CI_ARCHIVE_PATH" ]; then
    echo "Archive created at: $CI_ARCHIVE_PATH"
    echo "Archive size: $(du -sh "$CI_ARCHIVE_PATH" | cut -f1)"
fi

echo "=== Post Xcodebuild Complete ==="
```

## Environment Variables Reference

### Build Context

| Variable | Example | Description |
|----------|---------|-------------|
| `CI` | `TRUE` | Always TRUE in Xcode Cloud |
| `CI_WORKFLOW` | `PR Validation` | Workflow name |
| `CI_XCODEBUILD_ACTION` | `build`, `test`, `archive` | Current action |
| `CI_XCODEBUILD_EXIT_CODE` | `0`, `65` | Exit code (post script only) |
| `CI_BUILD_NUMBER` | `42` | Auto-incrementing number |
| `CI_BUILD_ID` | `abc123` | Unique build identifier |

### Source Control

| Variable | Example | Description |
|----------|---------|-------------|
| `CI_BRANCH` | `main` | Branch being built |
| `CI_TAG` | `v1.0.0` | Tag (if tag-triggered) |
| `CI_COMMIT` | `a1b2c3d4...` | Full commit SHA |
| `CI_PULL_REQUEST_NUMBER` | `42` | PR number (if PR-triggered) |
| `CI_PULL_REQUEST_SOURCE_BRANCH` | `feature/xyz` | PR source branch |
| `CI_PULL_REQUEST_TARGET_BRANCH` | `main` | PR target branch |

### Project

| Variable | Example | Description |
|----------|---------|-------------|
| `CI_PRODUCT` | `MyApp` | Product name |
| `CI_XCODE_PROJECT` | `MyApp.xcodeproj` | Project file |
| `CI_XCODE_SCHEME` | `MyApp` | Scheme being built |
| `CI_PRODUCT_PLATFORM` | `iOS` | Target platform |
| `CI_TEAM_ID` | `ABCDE12345` | Apple Developer Team ID |
| `CI_APP_STORE_ID` | `1234567890` | App Store app ID |

### Paths

| Variable | Example | Description |
|----------|---------|-------------|
| `CI_PRIMARY_REPOSITORY_PATH` | `/Volumes/...` | Repo root |
| `CI_DERIVED_DATA_PATH` | `/Volumes/.../DerivedData` | Build artifacts |
| `CI_ARCHIVE_PATH` | `/Volumes/.../Archive.xcarchive` | Archive location |
| `CI_RESULT_BUNDLE_PATH` | `/Volumes/.../ResultBundle.xcresult` | Test results |
| `CI_WORKSPACE_PATH` | `/Volumes/...` | Workspace path |

### Custom Environment Variables

Set custom secrets in Xcode Cloud settings (App Store Connect or Xcode):

```bash
# Accessing custom environment variables/secrets
echo "API Key: $MY_API_KEY"           # Set in Xcode Cloud settings
echo "Slack URL: $SLACK_WEBHOOK_URL"  # Set as secret
```

## Common Script Patterns

### Conditional by Branch

```bash
#!/bin/bash
case "$CI_BRANCH" in
    "main")
        echo "Production build"
        export API_BASE_URL="https://api.myapp.com"
        ;;
    "develop")
        echo "Development build"
        export API_BASE_URL="https://staging-api.myapp.com"
        ;;
    *)
        echo "Feature branch build"
        export API_BASE_URL="https://dev-api.myapp.com"
        ;;
esac
```

### Skip Steps for Speed

```bash
#!/bin/bash
# Skip heavy steps for PR builds
if [ -n "$CI_PULL_REQUEST_NUMBER" ]; then
    echo "PR build — skipping non-essential steps"
    exit 0
fi

# Only run for archive builds
if [[ "$CI_XCODEBUILD_ACTION" != "archive" ]]; then
    echo "Not an archive build — skipping"
    exit 0
fi
```

### Write Info.plist Values

```bash
#!/bin/bash
# ci_pre_xcodebuild.sh
PLIST_PATH="$CI_PRIMARY_REPOSITORY_PATH/MyApp/Info.plist"

if [ -f "$PLIST_PATH" ]; then
    /usr/libexec/PlistBuddy -c "Set :CFBundleVersion $CI_BUILD_NUMBER" "$PLIST_PATH"
    echo "Updated build number to $CI_BUILD_NUMBER"
fi
```

## Anti-Patterns

### ❌ Non-Executable Scripts

```bash
# ❌ Script won't run — not executable
# Fix: git update-index --chmod=+x ci_scripts/ci_post_clone.sh
```

### ❌ Using `exit 1` for Non-Fatal Errors

```bash
# ❌ Bad — Fails entire build for optional step
swiftlint || exit 1

# ✅ Good — Warn but don't fail
swiftlint || echo "SwiftLint warnings found (non-blocking)"
```

### ❌ Long-Running Post-Clone Scripts

```bash
# ❌ Bad — Installing everything in post_clone
brew install imagemagick ffmpeg python3 ruby node

# ✅ Good — Only install what's actually needed
brew install swiftlint  # Only essential tools
```
